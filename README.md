# Discord_Traders
Follow discord trader to see who has better performance 

## Starting the app (Windows)

1. Make sure Python 3 is installed (https://www.python.org/downloads/).
2. Double-click `start.bat` in this folder.
3. Wait for the app to open in your web browser.

The first run sets up everything automatically and may take a minute or two;
later runs start faster. Leave the black command window open while using the
app - closing it stops the app.
