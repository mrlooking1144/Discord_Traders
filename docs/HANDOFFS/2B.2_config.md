# Milestone

2B.2 — database/config.py

# Summary

Implemented a minimal, standard-library-only configuration module for the
Version 1 database. `database/config.py` defines a single dataclass,
`DatabaseConfig`, exposing exactly three fields: `db_path` (required, no
hardcoded or machine-specific default), `probable_duplicate_window_seconds`
(default `86400`, for later use by duplicate-detection logic), and
`busy_timeout_seconds` (default `5.0`, for later use when opening SQLite
connections). No connection logic, schema application, validation, or
environment/file-based configuration loading was added. No other modules,
tests, or documentation were touched during implementation.

- `database/config.py` (new)
